# Disease Prediction from Medical Data

This project trains a simple diabetes prediction model in VS Code using a structured medical dataset.

## How to run

1. Open the project folder in VS Code.
2. Open a terminal in the project root.
3. Run training:
   - "c:/Users/prava/Desktop/horizon techX/Disease prediction from medical data/.venv/Scripts/python.exe" data set/src/train.py
4. Run prediction:
   - "c:/Users/prava/Desktop/horizon techX/Disease prediction from medical data/.venv/Scripts/python.exe" data set/src/predict.py

## What the model does

- Loads the diabetes dataset
- Trains a Logistic Regression classifier
- Saves the trained model to the models folder
- Predicts whether a patient is likely to have diabetes
